Command Line

Handy Things To Know

  • cd: to change directories
  • mkdir: make a new directory/folder
  • pwd: to find our location
  • ls: gives you a list of the files in the current directory
  • cd ..: moves you up a directory
  • touch: to create an empty file such as a HTML file

Git Stuff

  1. be in the folder you want to commit
  2. git status
  3. git init: initializing a new Git repository
  4. git add then the file name e.g. hello-world.html
  5. git commit -m "make a comment about the file"
  6. git log: shows our Git history
  7. git remote add origin...
  8. git push origin master
