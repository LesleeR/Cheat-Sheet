Git Stuff

How to track, commit and push files to your GitHub account

  1. be in the folder you want to commit
  2. git status
  3. git init: initializing a new Git repository
  4. git add then the file name e.g. hello-world.html
  5. git commit -m "make a comment about the file"
  6. git log: shows our Git history
  7. git remote add origin https://github....
  8. git push origin master

git clone (URL):to copy from GitHub we get the URL for the remote repository

HTML Tags

  • !DOCTYPE html: tells the browser that this document contains HTML
  • ul: unordered list
  • ol: ordered list
  • title: sets the title for the web page browser tab.
  • body: this is the actual content of the page.

Inline elements

  • strong: bolds the text
  • em: italicize the text
  • a href="http://www.epicodus.com/">Epicodus< /a : the anchor tag a creates a link and the href attribute provides the web address the link will go to.
  • img src="kitten.jpeg" alt="a photo of a cute kitten": self closing tag that displays an image.

    link href="css/styles.css" rel="stylesheet" type="text/css"

    Rules

    We can make rules for blocks of text. For example to make all <h1> blocks of text have blue writing and aligned to the center we would write:

    h1 {

    color: blue;

    text-align: center;

    }

    Another example- make all <h2> blocks of text written in italics and centered by:

    h2 {

    font-style: italic;

    text-align: center;

    }

    Another example to make all the <p> blocks of text in a certain font we would type:

    p {

    font-family: sans-serif;

    }

    We can also apply rules to lists <ul> or <ol>:

    ul {

    font-size: 20px;

    line-height: 30px;

    }

    Classes

    To change just one or more of the <p> tags (or any other tag eg. h1,h2) but not ALL of the <p> within the page you create a class.For Example:

    <p> would become

    <p class="intro"> then on the style sheet you type the changes just for that tag:

    p.intro{

    width: 50%;

    font-style: italic;

    font-weight: bold;

    }

    You can also use the tag: div class="nameofclass" to edit large amounts of text to a certain style. Where the changes you make are to end you need to close the tag with /div

    We can also define a special style for individual tables!

    e.g. <table> would become

    <table id="t01">

    then on our CSS style sheet we can type:

    table#t01 {

    width: 100%;

    background-color: #f1f1c1;

    }
