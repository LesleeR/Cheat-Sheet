Git Stuff

How to track, commit and push files to your GitHub account

  1. be in the folder you want to commit
  2. git status
  3. git init: initializing a new Git repository
  4. git add then the file name e.g. hello-world.html
  5. git commit -m "make a comment about the file"
  6. git log: shows our Git history
  7. git remote add origin https://github.... To check if a remote has already been set up, just enter git remote
  8. git push origin master

git clone (URL):to copy from GitHub we get the URL for the remote repository

    Rules

    We can make rules for blocks of text. For example to make all <h1> blocks of text have blue writing and aligned to the center we would write:

    h1 {

    color: blue;

    text-align: center;

    }

    Another example- make all <h2> blocks of text written in italics and centered by:

    h2 {

    font-style: italic;

    text-align: center;

    }

    Another example to make all the <p> blocks of text in a certain font we would type:

    p {

    font-family: sans-serif;

    }

    We can also apply rules to lists <ul> or <ol>:

    ul {

    font-size: 20px;

    line-height: 30px;

    }

    Classes

    To change just one or more of the <p> tags (or any other tag eg. h1,h2) but not ALL of the <p> within the page you create a class.For Example:

    <p> would become

    <p class="intro"> then on the style sheet you type the changes just for that tag:

    p.intro{

    width: 50%;

    font-style: italic;

    font-weight: bold;

    }

    You can also use the tag: div class="nameofclass" to edit large amounts of text to a certain style. Where the changes you make are to end you need to close the tag with /div

    We can also define a special style for individual tables!

    e.g. <table> would become

    <table id="t01">

    then on our CSS style sheet we can type:

    table#t01 {

    width: 100%;

    background-color: #f1f1c1;

    }

    Bootstrap

    Bootstrap is a front-end framework that is essentially a really big CSS stylesheet. It allows developers to quickly set up professional looking websites with minimal effort.

    To integrate Bootstrap into our project we need to add one of these files (bootstrap.css or bootstrap.min.css ) to our project's css folder. Then we must link to it in our site's <head> tags, just like our own stylesheet:

    <head>

    <link href="css/bootstrap.css" rel="stylesheet" type="text/css" media="all"

    Containers

    Containers wrap our content nicely, and add padding that dynamically increases or decreases in size depending on the user's viewport.

    </head>

    </div>

    Jumbotron

    Jumbotrons are primarily used to add emphasis to content on a page. Often they are placed at the top of the page to act as a header.

    Just like a container, we can add a jumbotron by including a div with the jumbotron class wherever we'd like it to appear.

    <div class="jumbotron">

    Customising Styles in Bootstrap

    The Bootstrap website documents in detail how to use all of the styles it provides.

    Sometimes you might want to override some of Bootstrap's default styles with your own. To do this you link your custom stylesheet below Bootstrap's stylesheet in the <head> tags of your HTML. eg

    <head>

    <link href="css/bootstrap.css" rel="stylesheet" type="text/css">

    <link href="css/styles.css" rel="stylesheet" type="text/css">

    ...

    The second stylesheet will override the first stylesheet.
